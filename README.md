# 🌍 Life Expectancy Analysis – Machine Learning Project

This project analyzes **World Health Organization (WHO) life expectancy data** using data science and machine learning techniques.  
It explores the impact of **economic, demographic, and health-related factors** on life expectancy across different countries and builds models to predict life expectancy.  

---

## 📊 Project Overview

- **Domain:** Healthcare / Public Health Analytics  
- **Tools Used:** Python, Pandas, NumPy, Matplotlib, Seaborn, scikit-learn  
- **Project Type:** Exploratory Data Analysis (EDA) + Machine Learning  
- **Dataset Source:** WHO Life Expectancy Dataset (2000–2015)  
- **Difficulty Level:** Intermediate  

---

## 🧩 Problem Statement

Life expectancy is influenced by multiple socio-economic and healthcare-related factors.  
The goal of this project is to:  

- Analyze global life expectancy patterns.  
- Identify the most important features affecting life expectancy.  
- Build ML models to predict life expectancy using health, economic, and demographic indicators.  

---

## 🧠 Key Objectives

✔️ Perform **data cleaning** (handle missing values, duplicates, data types).  
✔️ Conduct **EDA** on global life expectancy data.  
✔️ Study the relationship between **GDP, schooling, healthcare spending, alcohol use, and life expectancy**.  
✔️ Apply **Linear Regression and Random Forest** models to predict life expectancy.  
✔️ Compare model performance and highlight key predictors.  

---

## 🗃️ Dataset Features

| Column              | Description |
|---------------------|-------------|
| `Country`           | Country name |
| `Year`              | Year of observation |
| `Life expectancy`   | Life expectancy in years |
| `Status`            | Developed / Developing |
| `Adult Mortality`   | Probability of dying between 15–60 years per 1000 population |
| `Infant deaths`     | Number of infant deaths per 1000 population |
| `Alcohol`           | Alcohol consumption per capita (liters) |
| `Expenditure`       | Health expenditure (% of GDP) |
| `Hepatitis B`       | Immunization coverage (%) |
| `Measles`           | Number of reported cases |
| `BMI`               | Average Body Mass Index |
| `Under-five deaths` | Deaths under age 5 per 1000 |
| `Polio`             | Polio immunization coverage (%) |
| `Schooling`         | Average years of schooling |
| `GDP`               | Gross Domestic Product (per capita) |

---

## 📈 Key Insights

- 🌍 **Developed countries** show higher life expectancy (avg ~79 years) compared to developing (~66 years).  
- 📉 **Adult mortality** and **infant deaths** strongly negatively correlate with life expectancy.  
- 🎓 **Schooling** and **GDP per capita** are highly positive predictors of life expectancy.  
- 🥂 Excessive **alcohol consumption** reduces life expectancy, but moderate consumption shows neutral effects.  
- 🤖 **Random Forest outperformed Linear Regression** in predictive accuracy.
